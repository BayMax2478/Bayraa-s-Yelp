ШАР НОМ — Системийн Шаардлагын Зохиомж

(Yelp / Yellow Book загварын үндэс дээр)

---------------------------------------------------------------------------------------------------------------------------

1. Төслийн зорилго

Монголын бүх төрлийн бизнес, үйлчилгээ, байгууллагуудыг нэг дор бүртгэж, хэрэглэгчид хайх, үнэлэх, сэтгэгдэл бичих, байршил харах, холбогдох боломжийг олгох онлайн платформ бий болгох. Энэхүү систем нь вэб болон мобайл хувилбартайгаар ажиллана.

---------------------------------------------------------------------------------------------------------------------------

 2. Гол функцууд

2.1 Хэрэглэгчийн функцууд

Бүртгэл / нэвтрэлт (email, Google, Facebook, Apple ID)
Байршил суурилсан хайлт
	→ Жишээ: “coffee shop near me”, “засварын газар Баянзүрх дүүрэг”
Бизнесийн дэлгэрэнгүй хуудас:
	→ Нэр, хаяг, зураг, ажиллах цаг, үнийн хүрээ, утас, вэбсайт, social линк
Үнэлгээ ба сэтгэгдэл бичих (1–5 од, текст, зураг хавсаргах)
Favorite / Save — дуртай газраа хадгалах
Report / Flag — буруу мэдээлэл, зохисгүй контентыг мэдээлэх
Directions / Map Navigation — Google Maps эсвэл MapBox integration

2.2 Бизнес эзэмшигчийн функцууд

Байгууллага бүртгэх / баталгаажуулах
Профайл засварлах (зураг, мэню, үйлчилгээний төрөл, хаяг гэх мэт)
Статистик үзэх — үзэлт, сэтгэгдэл, үнэлгээний дундаж
Хариу бичих (Review reply) — хэрэглэгчийн сэтгэгдэлд албан ёсоор хариулах

2.3 Админ функцууд

Хэрэглэгч, бизнес, контент хянах
Зохисгүй мэдээлэл устгах / Review report шийдвэрлэх
Категори, дүүрэг, салбарууд удирдах
Мэдээ, сурталчилгаа, баннерийн удирдлага

4. Өгөгдлийн загвар (анхны хувилбар)

User: id, name, email, password_hash, role, created_at, last_login
Business: id, owner_id, name, category_id, address, phone, website, latitude, longitude, open_hours, verified
Category: id, name, parent_id
Review: id, user_id, business_id, rating, comment, photo_url, created_at
Favorite: user_id, business_id
Report: id, user_id, business_id, reason, status

---------------------------------------------------------------------------------------------------------------------------

5. UI/UX хуудаснууд

Web

1. Нүүр / Хайлт
2. Бизнес жагсаалт
3. Бизнес дэлгэрэнгүй
4. Review бичих хуудас
5. Хэрэглэгчийн профайл
6. Админ самбар

Mobile App (Android/iOS)

1. GPS суурилсан ойр байгаа газрууд
2. Хайлтын хэсэг + ангиллын фильтр
3. Бизнес дэлгэрэнгүй хуудас
4. Review нэмэх (зураг хавсаргах боломжтой)
5. Saved / Favorite газар
6. Push мэдэгдэл

7. Non-functional Requirements

Uptime: ≥ 99%
Latency: Хайлтын хариу < 1 секунд (1000 бичлэг хүртэл)
Scalability: Cloud auto-scaling
Accessibility: Mobile responsive
Language: Монгол + Англи (i18n ready)

---------------------------------------------------------------------------------------------------------------------------

8. Хэрэглэгчийн түүх (User Stories)

US-01: Хэрэглэгч “coffee shop” гэж бичээд ойр байрлал дахь газруудыг олж чадна.
US-02: Хэрэглэгч үнэлгээ (4 од) өгч, зураг хавсаргаж чадна.
US-03: Бизнес эзэмшигч өөрийн профайлыг баталгаажуулж, мэдээлэл шинэчилнэ.
US-04: Админ буруу мэдээлэл устгаж чадна.
